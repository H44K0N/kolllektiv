# Vaskeliste for kollektiv

Enkel nettside for å holde oversikt over hvem som skal gjøre oppgaver og når.

Hver oppgave har intervall (hvor ofte det skal gjøres), oppgavebeskrivelse m.m. og når man
huker av på en utført oppgave, nullstilles den og roterer den som er ansvarlig for å utføre
oppgaven.

NB: Hard coded verdier for beboere i kollektivet ettersom den kun var til privat bruk.

Bruker local storage i nettleser (kjørte på samme nettbrett, teipet til kjøleskapet).
